\section{Conclusiones}
Se han alcanzado el objetivo de poder mostrar en una web de una forma visual el funcionamiento del algoritmo de búsqueda A estrella.

Se ha profundizado en los conocimientos sobre desarrollo web y se ha podido tener una toma de contacto muy buena con las tecnologías web más modernas de la actualidad.

Durante el desarrollo del proyecto se han aplicado conocimientos adquiridos a lo largo de la realización del máster.

Gracias al uso de las diferentes técnicas de desarrollo anteriormente mencionadas, se han mejorado notablemente las capacidades de desarrollo software del alumno.

\section{Líneas de trabajo futuras}
En este apartado listaremos las ideas que podrían ser desarrolladas en futuras versiones de este proyecto.

\begin{itemize}
\item \textbf{Añadir más algoritmos}: Podrían añadirse más algoritmos de búsqueda usando la base que ya está creada.

\item \textbf{Crear usuarios para que cada uno tenga sus propias búsquedas}: Ahora mismo no existen usuarios que puedan tener sus propios ejercicios asociados a sus cuentas, por ello cualquier visitante de la web puede ver todos los ejercicios creados. Estaría bien poder tener usuarios con sus propios ejercicios asociados.

\item \textbf{Comparación de algoritmos}: Una vez que se tengan más algoritmos añadidos a la aplicación podría añadirse una opción para comparar cómo funcionan y poder visualizar sus diferencias a la hora de actuar.

\item \textbf{Aplicación móvil}: Cuando se realizó esta aplicación ya se tenía en mente esta opción y por ello esta es una de las líneas futuras de trabajo con más posibilidades de llevarse a cabo, ya que el servidor está pensado para que sea fácil de acceder a los recursos. 

\item \textbf{Incluir juegos para ver como se resuelven}: Actualmente la visualización de la búsqueda es un poco abstracta usando tan solo información muy teórica para la visualización de los nodos. Podría intentarse incluir juegos estilo: sudoku, ocho puzzle, sokoban, etc, para que se viera de una forma más fácil y práctica el funcionamiento de los algoritmos.

\item \textbf{Mejorar la manera de introducir los datos}: Podría usarse algún sistema más visual para añadir los nodos, por ejemplo, simplemente pulsando en la zona del gráfico que se añadiera un nuevo nodo y que pudiera pinchar encima de el para añadir sus datos. 
\end{itemize}
