\resumen{%
El objetivo del proyecto es mostrar de una forma visual y sencilla como funciona el algoritmo de búsqueda en espacio de estados A estrella. Para ello hemos creado una página web en la que podemos introducir los distintos nodos que conforman el grafo del espacio de estados, y automáticamente mostraremos el árbol de búsqueda de forma visual y una tabla con todos los pasos que ha ido siguiendo el algoritmo para resolver el problema.

Además, para facilitar la comprensión del algoritmo y cómo éste se resuelve, podremos ir viendo paso a paso que se ha ido haciendo para obtener el resultado, en una tabla, y mostrando en el diagrama que nodos se están analizando de una manera más visual y fácil de comprender.

Esta herramienta pretende ser útil para todos aquellos estudiantes que pretendan comprender el funcionamiento de este algoritmo y poder comprobar que los ejercicios que realizan de forma manual están correctamente resueltos y en caso de no ser así poder ver en que punto se encuentra el error, mejorar la compresión que tienen del algoritmo y el proceso de aprendizaje.

Con la realización de este proyecto se pretende también poner en práctica las técnicas más populares del desarrollo de software (Scrum, los principios SOLID \footnote{SOLID es un acrónimo mnemónico introducido por Robert C. Martin (\en{Single responsibility, Open-closed, Liskov substitution, Interface segregation and Dependency inversion})}, TDD \footnote{Siglas en inglés de \en{test driven development}, desarrollo dirigido por pruebas}, BDD \footnote{Siglas en inglés de \en{behaviour driven development}, desarrollo dirigido por comportamiento}, etc), además de los conocimientos adquiridos durante la realización del máster y aprender a usar las herramientas de desarrollo web más en boga (JavaScript, Ruby, HTML, CSS).

En esta memoria se explicará el proceso de desarrollo que se ha llevado a cabo para la realización de este proyecto. En primer lugar se detallan los objetivos del proyecto, después se explicarán los conceptos teóricos más importantes para la comprensión de la memoria. Se comentarán las distintas técnicas y herramientas utilizadas para su desarrollo. Los aspectos más relevantes del proyecto, así como las dificultades encontradas. También se expondrán las conclusiones obtenidas y las líneas de trabajo que se podrían abordar en un futuro.

Se incluyen una serie de anexos que explicarán diversos aspectos del desarrollo, el primero abordará los aspectos relativos a la planificación del proyecto y su viabilidad, en el segundo se especificarán los requisitos del proyecto a realizar, el tercero contendrá las especificación de diseño para la implementación, el cuarto incluirá la documentación técnica de programación y por último se mostrará la ayuda de usuario

}
\descriptores{Algoritmo, A estrella, web, búsqueda en espacio de estados}

\abstract{%
The aim of the project is to show in a visual and easily way how the search algorithm A star works. So we've created a web page where you can enter the different nodes that made the graph of the state space, and automatically show the search tree visually and a table of all the steps you have been following to solve the algorithm. 

In addition, to facilitate understanding the algorithm and how it is resolved, we'll see what decisions has been taken to get the solution, step by step. Also the result has been showed in a table and in a graph for facilitate understanding. 

This tool will pretending to be useful for those students who wish to understand the operation of this algorithm and to verify that the exercises are performed manually are correctly resolved or otherwise to see at what point are error, improving the compression of the algorithm and the learning process. 

With this project, I intended to put in practice the most popular software development techniques (Scrum, the SOLID principles, TDD, BDD, etc), in addition I try to use all the knowledge gained through the realization of the master and also learn to use web development tools in vogue like JavaScript, Ruby, HTML and CSS.
}
